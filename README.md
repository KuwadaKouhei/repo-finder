# github-searcher

## AI利用ポイント

このプロジェクトでのAIの利用範囲は以下の基準で行う。

- 人間は思想の決定や判断、設計やレビューなど責任を伴うタスクを行う
- AIは提案や調査、人間の設計に基づいたコードの実装など責任が伴いにくい部分を任せる。

具体的には以下のような作業の振り分け方を行っている。

### 人間が行うタスク

- 設計思想・テスト思想の決定
- 要件定義のまとめ・決定
- AIが行ったタスクのレビュー・確認

### AIが行うタスク

- 要件定義の提案
- Mermaid図の作成
- Github APIの仕様調査(/docs/GITHUB_API.md の作成)
- Next.js v16の設計思想の調査
- コードの実装
